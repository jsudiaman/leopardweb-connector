# LeopardWeb Connector
Imports Wentworth Institute of Technology course schedules into Google Calendar.

## Dependencies
- Python 3.7
- [Pipenv](https://pipenv.readthedocs.io)

## How to Run
```
pipenv install
pipenv run python googlecalendar.py
```
